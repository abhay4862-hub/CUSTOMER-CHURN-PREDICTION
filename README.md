# 🧠 Customer Churn Prediction — Streamlit App

A complete ML-powered web app for predicting customer churn, built with Streamlit.

---



## 📱 App Pages

| Page | Description |
|---|---|
| 🏠 Home & Predict | Enter single customer details → get instant churn prediction |
| 📊 EDA | Explore dataset with distributions, boxplots, heatmap |
| 🤖 Model Performance | Confusion matrices, ROC curves, metrics comparison |
| 🔵 Clustering | K-Means cluster profiles, PCA scatter, elbow method |
| 📁 Batch Prediction | Upload CSV → predict churn for many customers at once |

---

## 🤖 Models Included

- Logistic Regression  
- Decision Tree  
- **Random Forest** ⭐ (Best AUC = 1.00)  
- Gradient Boosting  

---

## 📊 Dataset

- **440,833** customer records  
- **11** features: Age, Gender, Tenure, Usage Frequency, Support Calls,  
  Payment Delay, Subscription Type, Contract Length, Total Spend, Last Interaction  
- **Target**: Churn (1=Yes, 0=No)
